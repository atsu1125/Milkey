Groundpolis-Milkey
======================================================

- [日本語](README-ja.md)
- English

**A forever evolving, interplanetary microblogging platform.**

Groundpolis is a [Misskey](https://github.com/syuilo/misskey) 12.69.0 based microblogging platform.

Milkey is the beta edition of Groundpolis.
