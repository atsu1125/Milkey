**進化し続ける、洗練された、ミニブログプラットフォーム。**

Groundpolis (通称グランポリス)は、 [Misskey](https://github.com/misskey-dev/misskey)12.69.0 をベースに多くの改善や機能拡張を施したミニブログソフトウェアです。

そんなGroundpolisのNightly版がMilkeyです。試験的な新機能をいち早く体験できます。
